API Configuration

How to Use

1. Configure API Token: Enter your API token above and click "Save Token"

2. Visit Facebook Ad Library: Go to https://www.facebook.com/ads/library/

3. Manage Tags: Use the tag dropdowns next to each ad to add, remove, or select tags

4. Save Ads: Click "Save ad" to save ads with their selected tags

Note: Tag management is now handled directly in the Facebook Ad Library interface for easier access.
